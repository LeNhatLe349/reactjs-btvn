FLEX

This is shorthand for flex-grow, flex-shrink and flex-basis combined. The sencond and third paraments(flex-shrink and flex-basis) are oprional. Default is 0 1 auto

FLEX GROW

This denfines the ability for a flex item to grow if necessary, It accept a unitless value that servers as a proportion. It dictates what amount of the available space inside the flex container the item should take up.

FLEX SHRINK

This defines the ablity for a flex item to shrink if necessary. Negative numbers are invalid.
